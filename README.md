# MM-GBSA
Jupyter notebook based tutorial on protein-ligand binding energy prediction by MM-GBSA, prepared for CompBioAsia 2026

## Instructions

Clone this repo then:

* if you are NOT using the CBA Jupyterhub, execute `run_notebook.sh` to install required dependencies and launch the notebook.
* if you ARE using the CBA Jupyterhub than all requirements have already been installed for you, just open `MM-GBSA.ipynb`
